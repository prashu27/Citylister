# Citylister
code assessment 

## Pre-requisite

### Node js
### Java 11
### Gradle

## Steps to run the project locally : 

## Backend

### Using CMD

1. cd backend
2. ./gradlew bootRun

### Using Intellij

1. Import the project as Gradle project
2. Run Gradle Build
3. Run the main method (CitylisterApplication class)


## Frontend

In the project directory, you can run:

### cd frontend/

and then

### `npm start`
